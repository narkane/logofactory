CMS2CMS Connector

== Description ==
CMS2CMS Connector Plugin is designed to make your website migration become an effortless, flawless and automated process. This plugin establishes the connection between your Existing website built on any platform and the New WordPress site. It also works vice versa, in case you want to migrate your content from WordPress CMS.

= Major Advantages of Migration with CMS2CMS =
* Free Demo Migration provides with a possibility to observe the migration outcome beforehand;
* The average migration time is from several minutes to several hours. Timing depends on how many content items you’re going to move from your Existing website to the New one;
* Migrate the content you want. Choose how you want it to be and look on your New website  (Data ‘Mapping’ option);
* Complete data migration price depends on the content volume you’re about to migrate.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS Connector”.
5. Register an account or sign in to the existing one.
6. Click on the “Proceed” button. You’ll be redirected to the CMS2CMS website to proceed with your migration.
